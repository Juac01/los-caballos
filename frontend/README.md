# Frontend - Los Caballos

Aplicação Angular responsável pela interface do usuário e comunicação com a API backend.

---

## Tecnologias

- Angular
- TypeScript
- HTML / CSS

---

## Funcionalidades

- Tela de login
- Formulários para operações CRUD
- Validação de dados no cliente
- Integração com a API

---

## Como executar

```bash
cd frontend
npm install
ng serve
```
