# Backend - Los Caballos

API desenvolvida com Spring Boot para gerenciamento de dados e regras de negócio.

Desenvolvido utilizando IntelliJ IDEA.

---

## Tecnologias

* Java
* Spring Boot
* Maven
* H2 Database

---

## Funcionalidades

* Autenticação de usuários
* Criptografia de senha com BCryptPasswordEncoder
* CRUD completo
* Validação de dados (CPF, CNPJ, etc.)

---

## Banco de dados

* H2 Database (em memória)
* Utilizado para desenvolvimento e testes
* Executado junto com a aplicação na porta 8080

---

## Arquitetura

* MVC
* Controllers
* Services
* Models

---

## Execução

cd backend
./mvnw spring-boot:run
